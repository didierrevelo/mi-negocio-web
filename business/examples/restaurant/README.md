# La Cucina - Restaurant Platform

Sistema de restaurante completo con **Node.js**, **TypeScript** y **MongoDB**.

## Tech Stack

- **Backend:** Node.js + Express + TypeScript
- **Base de Datos:** MongoDB + Mongoose
- **Frontend:** HTML5 + CSS3 + JavaScript

## Instalación

```bash
cd backend
npm install
npm run dev
```

## API Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | /api/menu | Listar menú |
| POST | /api/menu | Crear plato |
| GET | /api/reservations | Listar reservaciones |
| POST | /api/reservations | Crear reservación |
| POST | /api/orders | Crear pedido |
